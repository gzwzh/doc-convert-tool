## 📊 技术细节

### 核心修改

```python
# 修复前 ❌
ppt_app.Visible = 0  # 导致RPC错误

# 修复后 ✅
ppt_app.Visible = 1  # 必须可见
ppt_app.WindowState = 2  # 最小化到任务栏
```

### 修改原理

- COM接口需要窗口消息循环
- 完全隐藏会停止消息循环
- 最小化保持消息循环运行
- 窗口不占用屏幕空间
