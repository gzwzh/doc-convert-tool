# 格式转换小程序 Demo

这是一个使用微信原生小程序开发的 Demo，用于演示如何与本地 Python 后端进行交互。

## 目录结构
- `pages/index/`: 首页代码
- `project.config.json`: 小程序项目配置文件
- `app.json`: 全局配置

## 快速开始

1. **启动后端服务**
   请确保 `backend/main.py` 正在运行。
   ```bash
   python backend/main.py
   ```
   后端默认运行在 `http://0.0.0.0:8002`。

2. **导入开发者工具**
   - 打开微信开发者工具
   - 选择“导入项目”
   - 选择本文件夹 (`miniprogram_demo`)
   - AppID 选择“测试号”或“游客模式”

3. **配置本地网络**
   - 在开发者工具顶部菜单栏，点击“详情” -> “本地设置”
   - 勾选 **“不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书”**

4. **真机调试**
   - 确保手机和电脑在同一局域网（连接同一个 WiFi）
   - 在电脑终端运行 `ipconfig` (Windows) 或 `ifconfig` (Mac/Linux) 查看局域网 IP
   - 在小程序界面输入框中填入该 IP，例如 `http://192.168.1.5:8002`

## 功能
目前实现了 PDF 转 Word (docx) 的基础流程：
1. 选择文件 (支持聊天文件)
2. 上传并转换
3. 下载并预览结果
