# 📄 DocFlow — 文档转换工具小程序  
## 🧾 Markdown 原型文档 & Figma 设计规范（v1.0）

## 🏠 二、首页（Home Page）

### 1. 导航栏（Top App Bar）
- 高度：48px
- 左侧：Logo（图标 + “DocFlow”）
  - 图标：24×24px 蓝色“D”字母（无衬线体）
  - 文字：16pt, `#1E293B`
- 右侧：历史记录图标（时钟图标，24×24px，`#64748B`）

### 2. 欢迎区（Hero Section）
- 标题：`轻松转换任何文档格式`  
  - 字号：18pt, Semibold, 居中，上边距 32px
- 副标题：`支持 PDF、Word、Excel、PPT、图片等 20+ 格式`  
  - 字号：14pt, `#64748B`, 居中，上下间距 8px

### 3. 主操作按钮（Upload Button）
- 容器：居中，宽高 100px，圆角 50%
- 背景：线性渐变 `to bottom right, #3B82F6 → #2563EB`
- 内部图标：“+” 或上传箭头（白色，28px）
- 下方文字：`点击上传文件`（14pt, `#64748B`, 居中，距按钮 12px）

### 4. 快捷转换入口（Horizontal Scroll Cards）
- 标题：`常用转换`（左对齐，16pt, `#1E293B`, 上边距 32px）
- 容器：横向滚动区域（高度 110px）
- 单个卡片：
  - 尺寸：80px × 90px
  - 背景：`#FFFFFF`
  - 圆角：12px
  - 阴影：轻微投影
  - 内容：
    - 图标（24×24px，主色 `#2563EB`）
    - 文字（两行，12pt，居中，如 “PDF → Word”）

> 示例卡片（6个）：
> 1. PDF → Word  
> 2. 图片 → PDF  
> 3. Word → PDF  
> 4. Excel → CSV  
> 5. PPT → PDF  
> 6. 扫描件 OCR（带扫描仪图标）

### 5. 最近文件列表（Recent Files）
- 标题：`最近文件`（左对齐，16pt）
- 列表项（最多显示 5 条）：
  - 高度：60px
  - 左侧：文件类型图标（PDF/DOC/XLS/PNG，24×24px）
  - 中间：
    - 文件名（14pt, `#1E293B`）
    - 状态标签（如“已转换”，绿色 `#10B981`，12pt）
  - 右侧：下载图标（云+向下箭头，20×20px，`#64748B`）

### 6. 底部 Tab 栏（Bottom Navigation）
- 固定底部，高度 56px
- 4 个选项（图标 + 文字，10pt）：
  - 首页（🏠，激活态蓝色）
  - 转换（🔄）
  - 我的文件（📁）
  - 我的（👤）

## 🔄 三、转换页（Convert Page）

> 用户从首页点击“上传”后进入此页

### 流程：
1. **上传文件** → 显示文件缩略图 + 文件名
2. **选择输出格式**（下拉菜单或网格）
   - 例如：输入为 PDF，可选输出：Word / JPG / TXT / Excel
3. **高级选项（可折叠）**
   - 页面范围（全部 / 自定义）
   - 是否压缩（开关）
   - OCR 识别（仅图片/PDF 扫描件）
4. **转换按钮**（固定底部）
   - 文字：“开始转换”
   - 宽度：100%，高度 56px，背景主色，白色文字

## ⚙️ 六、交互说明（Micro-interactions）

| 场景                     | 反馈方式                     |
|--------------------------|------------------------------|
| 上传成功                 | 轻微震动 + “文件已加载”提示 |
| 转换完成                 | 弹出 Toast：“✅ 转换成功！” + 下载按钮 |
| 网络错误                 | 红色提示条 + “重试”按钮     |
| 空状态（无文件）         | 插画 + “暂无文件，快去转换吧！” |
